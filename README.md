# Risk-analysis-and-defense-prioritization-

## Overview
Risk Analysis and Defense Prioritization is a cybersecurity project that helps identify, analyze, and prioritize security risks in a network or system. The project focuses on evaluating vulnerabilities, analyzing attack paths, and selecting the most effective defenses within a limited budget.

This system assists organizations and researchers in improving cybersecurity decision-making by ranking vulnerabilities and recommending optimized defense strategies.

---

## Features
- Vulnerability Risk Analysis
- Attack Path Analysis
- Risk Scoring and Ranking
- Budget-Constrained Defense Optimization
- Graph-Based Network Visualization
- CSV File Input Support
- Security Gain Calculation
- Optimal Defense Selection
